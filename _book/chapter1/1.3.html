快速开始

快速开始

本文将从现有的项目模板，让你快速运行一个VUE-SRR项目。

环境准备

  • 操作系统：支持 macOS，Linux，Windows
  • 运行环境：NodeJS，建议选择 LTS 版本，最低要求 8.x。

初始化

直接克隆模板项目，快速生成项目（v2会使用脚手架）

$ git clone https://github.com/mopacha/vue-ssr-admin.git
$ npm i

启动项目:

$ npm run dev
$ open http://localhost:3006
